# genOMFF
genOMFF: A tool for generating Gromacs files for the OMFF force field

1. Prepare the corresponding .gro file and table_list.dat.The pOMFF folder is placed in the working directory.
2. In GenpOMFF.py, specify the names and the number of the molecules included in the system.
3. Run GenpOMFF.py.
